05 — Conexiones entre componentes
Browser → Vercel CDN
HTTPS desde cualquier dispositivo. React SPA se sirve desde el edge más cercano al usuario. Sin datos personales en esta capa.
React → Cloud Run (API calls)
React llama a VITE_API_URL/api/* en Cloud Run vía HTTPS. Nunca habla directo a Firestore desde el browser.
Cloud Run → Firestore
Firebase Admin SDK con Service Account nativa GCP. Sin archivo JSON de credenciales. Conexión interna dentro de la misma región southamerica-west1.
Cloud Run → Cloud Storage
Node.js sube archivos y genera Signed URLs de 72h para descarga directa. El archivo va del ciudadano al Storage sin pasar por Cloud Run.
DPO Browser → Firebase Auth → Cloud Run
El DPO hace login en Firebase Auth desde el browser, recibe un JWT, y lo envía en cada request a Cloud Run. Cloud Run lo verifica con Admin SDK.
Cloud Run → Resend (email)
Llamada HTTPS asíncrona saliente. Node.js dispara el email y no espera respuesta para liberar el request del ciudadano. API key guardada en Secret Manager.
